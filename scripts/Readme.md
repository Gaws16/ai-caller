Това са powershell команди,които с 1 клик можеш да активираш някаква команда или да seed-неш в Supabase някакъв script.

Прави си powershell команди,за да си улесниш живота като:

- Reset database 1 click run
- Seed users 1 click run
- Delete seed users and any leftover data 1 click run
- Notification triggers 1 click run
- Stripe webhook listener setup 1 click run
